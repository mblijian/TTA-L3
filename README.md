# TTA-L3
This is for L3T2 students.

## Python Dictionary Learning Game

Learn Python dictionaries through an engaging adventure game! This repository contains three files designed to teach dictionary concepts to beginners:

### Files

1. **dictionary_basics_practice.py** - Start here!
   - Interactive exercises covering dictionary fundamentals
   - 9 exercises from basic to advanced concepts
   - Mini project: Contact Book application
   - Perfect for absolute beginners

2. **dictionary_adventure_game.py** - The main game!
   - Engaging RPG-style adventure game
   - Learn dictionaries through gameplay
   - Features: character creation, combat, inventory system, and more
   - Survive 10 turns and maximize your score!

3. **DICTIONARY_GAME_GUIDE.md** - Complete learning guide
   - Detailed explanations of all dictionary concepts
   - How to play instructions
   - Code examples from the game
   - Practice challenges

### Getting Started

1. **Learn the basics:**
   ```bash
   python dictionary_basics_practice.py
   ```

2. **Play the adventure game:**
   ```bash
   python dictionary_adventure_game.py
   ```

3. **Read the guide:**
   - Open DICTIONARY_GAME_GUIDE.md for detailed explanations

### What You'll Learn

- Creating and using dictionaries
- Accessing and updating dictionary values
- Adding and removing items
- Nested dictionaries
- Looping through dictionaries
- Dictionary methods and best practices

### Requirements

- Python 3.x
- No external libraries required!

Have fun learning Python dictionaries!
